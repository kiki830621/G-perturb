The problem

Dependability, not effect size

A large effect measured in one guide, one donor, or a pseudobulk of a handful of cells can be an artifact. For target prioritization — where every false lead costs a validation experiment — dependability matters as much as magnitude.

The method

A measurement, decomposed

Each perturbation effect is a measurement in a crossed design. Two facets are random (generalize across them); one is fixed (consistency, not sampling).

TargetGuide · random Donor · randomCondition · fixed
R_dep,t = 1 / ( 1 + (1−R^guide)/R^guide + (1−R^donor)/R^donor ) // = correlation of two independent redos
